Contact Form with Validation

Input Data

A contact form with server-side validation for name, email, and message fields.

PHP 
<?php
$errors = [];

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // Validate name
    $name = trim($_POST['name']);
    if (empty($name)) {
        $errors[] = "Name is required";
    }
    
    // Validate email
    $email = filter_input(INPUT_POST, 'email', FILTER_VALIDATE_EMAIL);
    if (!$email) {
        $errors[] = "Valid email is required";
    }
    
    // Validate message
    $message = trim($_POST['message']);
    if (empty($message) || strlen($message) < 10) {
        $errors[] = "Message must be at least 10 characters";
    }
    
    // If no errors, process the form
    if (empty($errors)) {
        echo "<p style='color:green;'>Thank you, $name! Your message has been sent.</p>";
    } else {
        echo "<ul style='color:red;'>";
        foreach ($errors as $error) {
            echo "<li>$error</li>";
        }
        echo "</ul>";
    }
}
?>

<form method="post">
    Name: <input type="text" name="name"><br>
    Email: <input type="email" name="email"><br>
    Message: <textarea name="message"></textarea><br>
    <button type="submit">Send</button>
</form>

Expected Output:

Valid submission shows: "Thank you, Ahmed! Your message has been sent." Invalid input shows error list.

Student Grade Processor

Loops

Process an array of student grades using foreach loop to calculate statistics.

PHP 
<?php
// Array of students with their grades
$students = [
    ["name" => "Ahmed", "grade" => 95],
    ["name" => "Sara", "grade" => 88],
    ["name" => "Mohamed", "grade" => 76],
    ["name" => "Fatima", "grade" => 92],
    ["name" => "Ali", "grade" => 84]
];

echo "<h3>Student Grade Report</h3>";
echo "<table border='1' cellpadding='10' style='border-collapse: collapse;'>";
echo "<tr style='background: #3b82f6; color: white;'>";
echo "<th>Student Name</th><th>Grade</th><th>Letter</th><th>Status</th>";
echo "</tr>";

$totalGrade = 0;
$highestGrade = 0;
$lowestGrade = 100;
$passCount = 0;

// Process each student
foreach ($students as $student) {
    $name = $student['name'];
    $grade = $student['grade'];
    
    // Calculate statistics
    $totalGrade += $grade;
    if ($grade > $highestGrade) $highestGrade = $grade;
    if ($grade < $lowestGrade) $lowestGrade = $grade;
    
    // Determine letter grade
    if ($grade >= 90) {
        $letter = "A";
        $color = "green";
    } elseif ($grade >= 80) {
        $letter = "B";
        $color = "blue";
    } elseif ($grade >= 70) {
        $letter = "C";
        $color = "orange";
    } else {
        $letter = "F";
        $color = "red";
    }
    
    // Determine pass/fail
    $status = ($grade >= 60) ? "‚úì Pass" : "‚úó Fail";
    if ($grade >= 60) $passCount++;
    
    // Display row
    echo "<tr>";
    echo "<td>$name</td>";
    echo "<td style='text-align: center;'>$grade</td>";
    echo "<td style='color: $color; text-align: center;'><strong>$letter</strong></td>";
    echo "<td style='text-align: center;'>$status</td>";
    echo "</tr>";
}

echo "</table>";

// Calculate and display statistics
$average = $totalGrade / count($students);
$passRate = ($passCount / count($students)) * 100;

echo "<h4>Class Statistics</h4>";
echo "<p>üìä Average Grade: <strong>" . number_format($average, 2) . "</strong></p>";
echo "<p>üèÜ Highest Grade: <strong style='color: green;'>$highestGrade</strong></p>";
echo "<p>üìâ Lowest Grade: <strong style='color: red;'>$lowestGrade</strong></p>";
echo "<p>‚úÖ Pass Rate: <strong>" . number_format($passRate, 1) . "%</strong></p>";
?>

Expected Output:

A formatted table showing each student's name, numeric grade, letter grade, and pass/fail status, followed by class statistics.
