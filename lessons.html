How to Input Data in PHP

Learn different methods to receive and process user input in PHP applications.

1. GET Method

The GET method sends data through URL parameters. It's visible in the URL and suitable for non-sensitive data.

PHP 
<?php
// Receiving data from URL: example.php?name=Ahmed&age=20
$name = $_GET['name'];
$age = $_GET['age'];

echo "Name: " . $name . "<br>";
echo "Age: " . $age;
?>

2. POST Method

The POST method sends data in the request body. It's more secure and suitable for sensitive information.

PHP 
<?php
// Receiving data from form with POST method
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST['username'];
    $password = $_POST['password'];
    
    echo "Username: " . htmlspecialchars($username);
}
?>

3. HTML Form Input

HTML forms are the most common way to collect user input in web applications.

HTML + PHP 
<!-- HTML Form -->
<form method="post" action="process.php">
    <input type="text" name="email" placeholder="Email">
    <input type="text" name="message" placeholder="Message">
    <button type="submit">Send</button>
</form>

<?php
// process.php
$email = filter_input(INPUT_POST, 'email', FILTER_SANITIZE_EMAIL);
$message = htmlspecialchars($_POST['message']);
echo "Email: $email<br>Message: $message";
?>

Key Points

  • Always validate and sanitize user input to prevent security vulnerabilities
  • Use POST for sensitive data like passwords and personal information
  • GET is suitable for search queries and data that can be bookmarked

Basic Numeric Operators in PHP

Master arithmetic operations and mathematical calculations in PHP.

1. Basic Arithmetic Operators

PHP supports all standard arithmetic operations: addition, subtraction, multiplication, division, and modulus.

PHP 
<?php
$a = 10;
$b = 3;

$addition = $a + $b;        // 13
$subtraction = $a - $b;     // 7
$multiplication = $a * $b;  // 30
$division = $a / $b;        // 3.333...
$modulus = $a % $b;         // 1 (remainder)

echo "Addition: $addition<br>";
echo "Subtraction: $subtraction<br>";
echo "Multiplication: $multiplication<br>";
echo "Division: $division<br>";
echo "Modulus: $modulus";
?>

2. Assignment Operators

Assignment operators allow you to assign values and perform operations simultaneously.

PHP 
<?php
$x = 5;

$x += 3;  // $x = $x + 3; (now $x is 8)
$x -= 2;  // $x = $x - 2; (now $x is 6)
$x *= 4;  // $x = $x * 4; (now $x is 24)
$x /= 3;  // $x = $x / 3; (now $x is 8)
$x %= 5;  // $x = $x % 5; (now $x is 3)

echo "Final value: $x"; // Output: 3
?>

3. Increment and Decrement

These operators increase or decrease a variable's value by one.

PHP 
<?php
$counter = 5;

// Pre-increment: increment first, then use
echo ++$counter; // Output: 6

// Post-increment: use first, then increment
echo $counter++; // Output: 6 (then becomes 7)

// Pre-decrement: decrement first, then use
echo --$counter; // Output: 6

// Post-decrement: use first, then decrement
echo $counter--; // Output: 6 (then becomes 5)

echo $counter; // Output: 5
?>

Key Points

  • Division by zero will cause an error, always check before dividing
  • Use parentheses to control operation precedence
  • Modulus (%) is useful for checking even/odd numbers

Conditional Statements in PHP

Learn how to make decisions in your code using if, else, and switch statements.

1. If Statement

The if statement executes code only if a specified condition is true.

PHP 
<?php
$age = 18;

if ($age >= 18) {
    echo "You are an adult.";
}

// With comparison operators
$score = 85;
if ($score >= 90) {
    echo "Excellent!";
} elseif ($score >= 70) {
    echo "Good job!";
} else {
    echo "Keep trying!";
}
?>

2. Switch Statement

The switch statement is used to perform different actions based on different conditions.

PHP 
<?php
$day = "Monday";

switch ($day) {
    case "Monday":
        echo "Start of the work week!";
        break;
    case "Friday":
        echo "Almost weekend!";
        break;
    case "Saturday":
    case "Sunday":
        echo "It's the weekend!";
        break;
    default:
        echo "It's a regular day.";
}
?>

3. Ternary Operator

A shorthand way to write simple if-else statements in one line.

PHP 
<?php
// Syntax: condition ? value_if_true : value_if_false

$age = 20;
$status = ($age >= 18) ? "Adult" : "Minor";
echo $status; // Output: Adult

// Nested ternary
$score = 75;
$grade = ($score >= 90) ? "A" : 
         (($score >= 80) ? "B" : 
         (($score >= 70) ? "C" : "F"));
echo "Grade: $grade"; // Output: Grade: C
?>

Key Points

  • Use == for value comparison and === for value and type comparison
  • Always include 'break' in switch cases to prevent fall-through
  • Ternary operators are great for simple conditions but can be hard to read when nested

Loops in PHP

Master different types of loops to repeat code execution efficiently.

1. For Loop

The for loop is used when you know how many times you want to execute a block of code.

PHP 
<?php
// Basic for loop
for ($i = 1; $i <= 5; $i++) {
    echo "Number: $i<br>";
}

// Loop through an array
$fruits = ["Apple", "Banana", "Orange"];
for ($i = 0; $i < count($fruits); $i++) {
    echo "Fruit: " . $fruits[$i] . "<br>";
}

// Nested for loop (multiplication table)
for ($i = 1; $i <= 3; $i++) {
    for ($j = 1; $j <= 3; $j++) {
        echo "$i x $j = " . ($i * $j) . "<br>";
    }
}
?>

2. While Loop

The while loop executes a block of code as long as a specified condition is true.

PHP 
<?php
// Basic while loop
$counter = 1;
while ($counter <= 5) {
    echo "Count: $counter<br>";
    $counter++;
}

// While loop with condition check
$number = 10;
while ($number > 0) {
    if ($number % 2 == 0) {
        echo "$number is even<br>";
    }
    $number--;
}

// Do-while loop (executes at least once)
$x = 1;
do {
    echo "Value: $x<br>";
    $x++;
} while ($x <= 3);
?>

3. Foreach Loop

The foreach loop is specifically designed for iterating through arrays.

PHP 
<?php
// Foreach with indexed array
$colors = ["Red", "Green", "Blue"];
foreach ($colors as $color) {
    echo "Color: $color<br>";
}

// Foreach with associative array
$student = [
    "name" => "Ahmed",
    "age" => 20,
    "grade" => "A"
];
foreach ($student as $key => $value) {
    echo "$key: $value<br>";
}

// Foreach with modification
$numbers = [1, 2, 3, 4, 5];
foreach ($numbers as &$num) {
    $num = $num * 2; // Double each number
}
print_r($numbers); // [2, 4, 6, 8, 10]
?>

Key Points

  • Use 'break' to exit a loop early and 'continue' to skip to the next iteration
  • Foreach is the most efficient way to iterate through arrays in PHP
  • Always ensure your loop has a way to end to avoid infinite loops

How to Display Output in PHP

Learn different methods to display data and generate output in PHP.

1. Echo Statement

Echo is the most commonly used output statement in PHP. It can display multiple strings.

PHP 
<?php
// Simple echo
echo "Hello, World!";

// Echo with variables
$name = "Ahmed";
echo "Hello, " . $name . "!<br>";

// Echo with HTML
echo "<h2>Welcome to PHP</h2>";
echo "<p>This is a paragraph.</p>";

// Echo multiple values
echo "Value 1", " Value 2", " Value 3";

// Echo with calculations
$x = 5;
$y = 3;
echo "Sum: " . ($x + $y);
?>

2. Print Statement

Print is similar to echo but can only output one string and returns a value.

PHP 
<?php
// Basic print
print "Hello from print!<br>";

// Print with variables
$age = 20;
print "Age: $age<br>";

// Print returns 1
$result = print "This returns 1";
echo $result; // Output: 1

// Print in expressions
$condition = true;
$condition ? print "True" : print "False";
?>

3. Advanced Output Functions

PHP provides specialized functions for formatted output and debugging.

PHP 
<?php
// printf - formatted output
$name = "Ahmed";
$age = 20;
printf("Name: %s, Age: %d<br>", $name, $age);

// print_r - display array structure
$fruits = ["Apple", "Banana", "Orange"];
print_r($fruits);

// var_dump - detailed variable information
$number = 42;
var_dump($number);

// sprintf - return formatted string
$formatted = sprintf("Hello, %s! You are %d years old.", "Ahmed", 20);
echo $formatted;
?>

Key Points

  • Echo is faster than print and more commonly used
  • Use print_r() and var_dump() for debugging arrays and objects
  • Always escape output when displaying user input to prevent XSS attacks
